## Key Learnings

1. **Parametric Modeling**: Understanding how dimensions and constraints create flexible, modifiable designs.

2. **Workflow Efficiency**: Learning keyboard shortcuts and tool locations significantly speeds up the modeling process.

3. **Design Intent**: Thinking about how the model will be fabricated influences design decisions from the start.
